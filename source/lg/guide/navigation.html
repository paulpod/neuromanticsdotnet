Navigation

The old design uses a tab / sub-tab layout on two horizontal levels. Both these navigation layers are now in a simple vertical list, sub-lists appearing in a lighter font weight and indented. Metadata blocks to navigate to a tag/search, or a task, or a linked document or resource appear in a right hand column.

Main Navigation

For the specific set of default specimens, it is suggested the user be allowed to customise this list of 14 items, as it was considered unlikely an individual user would require all types available at all times. In addition, this scheme of navigation allows the sub-sub level as currently required for rodent strains/specimens/cages or any future requirement for this third depth of navigation.

Header

The header contains four main areas, each providing navigation at a smaller hierarchical layer of the 'account':

Labguru > User > Laboratory > Workspaces + Search

Each of these sections has a small number of direct links:

Labguru
Home - the main public site entrance
Help - a unified page with FAQ's, support documents, links to live-help webchat, etc.

User
Profile - a combined profile and settings page, user can edit
Messages - inter-team messaging
Logout immediate logout from shared machines (although short-ish session time reduces usefulness a bit)

Group
Members - clickable list of team members, links to profile pages, summary newsfeed and activity
Group tags - links to a tag cloud and then quick access to (a search really) to all tagged data within the group

Workspace
Name - select a different workspace from a drop down
Index - back to the main workspace chooser with big book icons and team listed under each one
