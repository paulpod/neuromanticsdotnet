Tags

Working out some consistent design solutions to the application of Tags on multiple page templates - user needs to see a contextual subset of tags, and in some places add new ones.

Tags contained as a popup list

Pros: Tidy, unobtrusive, can accommodate large lists, allows reuse anywhere

Cons: Possible selections are hidden, requires user input to reveal, very long lists may require clever JS scroller
